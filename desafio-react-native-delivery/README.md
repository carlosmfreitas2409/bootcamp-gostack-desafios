## 🚀 Sobre o Desafio

Nesse desafio, o objetivo foi continuar desenvolvendo uma nova aplicação, a GoMarketplace, só que dessa vez, a versão mobile para o client. Para então, praticar o que eu aprendi até agora no React Native junto com TypeScript, para criar um pequeno app para pedidos de comida.

Essa aplicação se conecta com uma Fake API, para prover dados sobre os produtos.

## 💿 Instalação

Para executar este projeto, você deve possuir o Node e o Yarn instalado para configurar todas as dependências.

```
- Clone o repositório:
$ git clone https://github.com/carlosmfreitas2409/bootcamp-gostack-desafios

- Entre no diretório:
$ cd desafio-react-native-delivery

- Para instalar as dependências:
$ yarn

- Execute a aplicação:
$ yarn start
$ yarn json-server server.json -p 3333

- Abra seu Android/IOS emulator
```
